# Sorority Recruitment Optimization System
A recruitment optimization platform designed to improve large-scale member-to-candidate matching using min-cost max-flow optimization and NLP-based compatibility scoring.

The system combines ranking workflows, semantic matching, and operational dashboards to support more efficient and data-driven recruitment coordination.

## Features
- Min-cost max-flow matching optimization
- NLP-based compatibility scoring
- Semantic tagging and fuzzy matching
- Streamlit dashboard interface
- Google Sheets API integration
- Real-time recruitment workflow management

## Tech Stack
- Python
- Streamlit
- Pandas
- Google Sheets API
- NLP / text processing
- Network optimization algorithms
